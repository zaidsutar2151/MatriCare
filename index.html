About the Project

MatriCare AI is a dual-module intelligent healthcare system designed to assist doctors and nurses during pregnancy and labor. It continuously monitors maternal parameters in real-time using sensors and provides AI-driven suggestions for potential complications.

The Problem

800+

Women die every day due to preventable pregnancy and childbirth complications (WHO, 2023)

94%

Of maternal deaths occur in low and lower-middle income countries

97 per 100K

India's maternal mortality ratio — still above the UN SDG goal of 70

75%

Hemorrhage, hypertension, and infection account for nearly 75% of all maternal deaths

60-70%

Of rural hospitals lack continuous monitoring systems for mothers during labor

Our Solution

MatriCare AI bridges the gap between real-time maternal monitoring and intelligent clinical decision support using two integrated modules:

Continuous Monitoring Module

Automatic
  • Collects data automatically from sensors
  • Tracks 8 vital parameters: BP (Sys/Dia), HR, Temp, SpO₂, RR, FHR, Toco
  • Processes live input through an ML model for instant status prediction (Stable / Moderate / Critical)
  • Displays data on a live dashboard using charts and color-coded alerts

Diagnostic & Suggestion Module

LLM-Powered
  • Accepts 19 manually entered maternal parameters
  • Classifies condition using an ML model
  • Applies an LLM trained on clinical rule data (from final data.xlsx)
  • Generates a detailed analysis including:
    • Future Complications
    • Basic Actions (for nurses)
    • Advanced Actions (for doctors)
